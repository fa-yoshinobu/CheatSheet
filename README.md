# CheatSheet

このリポジトリは、様々なソフトウェアやツールのショートカットキーやチートシートを管理するためのプロジェクトです。

## 📁 プロジェクト構造

```
CheatSheet/
├── KV_STUDIO/
│   ├── KV STUDIO ショートカットキー 20220813_初期形式.png
│   ├── KV STUDIO ショートカットキー 20220813_GPPQ.png
│   ├── KV STUDIO ショートカットキー 20220813.pdf
│   └── KV STUDIO ショートカットキー 20220813.odg
└── README.md
```

## 🎯 目的

- 開発効率を向上させるためのショートカットキーの整理
- チーム内での知識共有
- 新しいメンバーのオンボーディング支援

## 📂 ディレクトリ構成

### KV_STUDIO/
KV STUDIOのショートカットキーに関するファイルが含まれています。

**ファイル一覧：**
- `KV STUDIO ショートカットキー 20220813_初期形式.png` - 初期形式のショートカットキー画像
- `KV STUDIO ショートカットキー 20220813_GPPQ.png` - GPPQ形式のショートカットキー画像
- `KV STUDIO ショートカットキー 20220813.pdf` - PDF形式のショートカットキー
- `KV STUDIO ショートカットキー 20220813.odg` - LibreOffice Draw形式のソースファイル
